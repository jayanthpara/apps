# WebDesk OS - Detailed Master Prompt

Build a modern browser-based personal operating system called **WebDesk OS** using only **HTML, CSS, and Vanilla JavaScript** inside a single `index.html` file.

This is NOT a bookmark page and NOT a collection of hyperlinks.

The goal is to create a fully customizable personal productivity dashboard that feels like a lightweight operating system running inside a browser.

---

## Core Vision

The application should look and behave like a combination of:

* Windows 11
* macOS
* Notion
* Arc Browser Dashboard
* Android Home Screen
* Productivity Workspace

The user should be able to use the dashboard every day as their primary personal workspace.

The design should feel premium, modern, smooth, and highly customizable.
